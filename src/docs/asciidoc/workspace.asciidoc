== Workspace Recipes

The Workspace classes are in the *http://geoscript.github.io/geoscript-groovy/api/1.9.0/geoscript/workspace/package-summary.html[geoscript.workspace]* package.

A Workspace is a collection of Layer.  You can create, add, remove, and get Layers.  There are many
different kinds of Workspace in GeoScript including Memory, PostGIS, Directory (for Shapefiles), GeoPackage,
and many more.
